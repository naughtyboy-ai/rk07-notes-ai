Upload instructions (short):

  1. Create folders in your repo like notes/class12/maths/, notes/class11/physics/, etc.
  2. Upload PDFs into their respective folders (use GitHub → Add file → Upload files).
  3. If filenames have spaces or special chars, consider renaming to hyphen/slugs (recommended) — but the current HTML links use the original names you uploaded.
  4. Then enable Pages: Settings → Pages → Source: main branch → Save. Your site will appear at https://<username>.github.io/.

Tip: If a download doesn't work, check file name exactness and avoid leading/trailing spaces in filenames.
